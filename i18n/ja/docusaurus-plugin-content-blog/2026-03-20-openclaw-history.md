# OpenClaw の物語：Clawdbot から 25 万スターへ

すべての偉大なオープンソースプロジェクトの背後には、魅力的な物語があります。OpenClaw の旅 — 個人のサイドプロジェクトから 250,000 以上の GitHub Stars を持つ社会現象的プロジェクトへ — は、オープンソース精神の最高の体現です。

<!-- truncate -->

## 起源：Clawdbot（2024 年中頃）

物語は **Peter Steinberger** から始まります。Steinberger は iOS 開発コミュニティの著名人で、PSPDFKit（後の Nutrient）の創業者であり、シリアルアントレプレナーかつオープンソースの提唱者です。

2024 年中頃、Steinberger は個人的な実験プロジェクト **Clawdbot** を開始しました。当初のコンセプトはシンプルなものでした：複数のメッセージングプラットフォームに接続できる AI チャットボットフレームワークを構築すること。彼の目標は以下の通りです：

- 単一の統合インターフェースで複数のメッセージングプラットフォームを管理
- プラグインベースの機能拡張をサポート
- 技術者でなくても簡単に AI アシスタントをデプロイできるようにする

Clawdbot という名前は「Claw」（爪）と「Bot」を組み合わせたもので、ロブスターのイメージはここから生まれました。

## 進化：Moltbot（2024 年末）

プロジェクトが成長するにつれて、Steinberger は元の設計を大幅に見直す必要があることに気づきました。ロブスターが成長するために脱皮（Molt）するように、プロジェクトもまた「脱皮」を経験しました。

2024 年末、Clawdbot は **Moltbot** として再構築され、まったく新しいアーキテクチャが導入されました：

- **三層アーキテクチャ設計**：Gateway（接続層）→ Agent（プロキシ層）→ Skill（機能層）
- **WebSocket リアルタイム通信**：従来の REST ポーリング方式を置き換え
- **Skill Marketplace のプロトタイプ**：開発者が自作の Skill を共有可能に

Moltbot は小規模な開発者コミュニティで注目を集め、わずか数週間で GitHub Stars が 10,000 を突破しました。

## 転換点：OpenClaw への改名（2026 年 1 月）

2026 年 1 月、プロジェクトは最も重要なマイルストーンの一つを迎えました — **OpenClaw** への正式な改名です。

これは単なる名前の変更ではありませんでした。プロジェクトの方向性における根本的な転換を意味していました：

1. **完全なオープンソースガバナンス**：OpenClaw Foundation が設立され、コミュニティ主導のガバナンスを導入
2. **エンタープライズ機能**：RBAC（ロールベースアクセス制御）、監査ログ、SSO 統合を追加
3. **多言語サポート**：元の TypeScript/JavaScript に加えて、Python、Go、Rust、Java SDK を追加
4. **公式 Skill Marketplace**：13,000 以上のコミュニティ提供 Skill

改名後、プロジェクトの成長は驚異的でした。わずか 2 ヶ月で GitHub Stars は 50,000 から 150,000 へと急上昇しました。

## Peter Steinberger が OpenAI に参加（2026 年 2 月）

2026 年 2 月、オープンソースコミュニティを震撼させるニュースが飛び込んできました — **Peter Steinberger が OpenAI に参加**したのです。

Steinberger は OpenAI で「AI Agent インフラストラクチャ」に焦点を当てた新チームを率いることになりました。この決定はコミュニティで広く議論を呼びました：

> 「これは OpenClaw にとって何を意味するのか？」

Steinberger は発表の中で以下の点を明確にしました：

- OpenClaw は独立したオープンソースプロジェクトとして運営を続ける
- OpenClaw Foundation はすでに堅牢なコミュニティガバナンス体制を確立している
- 彼はアドバイザーとして OpenClaw の方向性に引き続き参加する

結果的に、この動きはむしろ OpenClaw により多くの注目とリソースをもたらしました。OpenAI もまた OpenClaw Foundation の企業スポンサーの一つとなりました。

## 25 万スターのマイルストーン（2026 年 3 月）

2026 年 3 月、OpenClaw は **250,000 GitHub Stars** を正式に突破し、GitHub 上で最も急成長しているオープンソースプロジェクトの一つとなりました。

この数字の背後にあるのは：

- **活発なコントリビューター**：世界中から 2,000 人以上のコントリビューター
- **Skill エコシステム**：生産性、開発ツール、コミュニティ管理など幅広い分野をカバーする 13,000 以上のコミュニティ Skill
- **プラットフォーム統合**：20 以上のメッセージングプラットフォーム（Slack、Discord、Teams、LINE、Telegram、WhatsApp など）をサポート
- **エンタープライズ採用**：数百の企業が本番環境で OpenClaw を使用
- **言語サポート**：5 つのプログラミング言語の公式 SDK

| 2025 年 | Moltbot の継続的な成長、コミュニティの拡大 |
| 2026 年 1 月 | OpenClaw に正式改名、Foundation を設立 |

## 今後の展望

OpenClaw の物語はまだ終わりません。コミュニティが積極的に開発中の機能には以下のものがあります：

- **OpenClaw Cloud**：ワンクリックデプロイの公式ホスティングサービス
- **Skill Studio**：ビジュアルな Skill 開発ツール
- **Agent Orchestration**：マルチ Agent コラボレーションフレームワーク
- **Edge Deployment**：エッジデバイスでの OpenClaw 実行をサポート

一人のサイドプロジェクトから世界的なオープンソース現象へ。OpenClaw の旅は一つのことを証明しています：**素晴らしいアイデアとオープンなコミュニティがあれば、世界を変えることができる。**
