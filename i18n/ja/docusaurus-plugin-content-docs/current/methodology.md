## Top 50 スキル評価基準

各スキルは8つの次元で10点満点で評価され、総合スコアは最大80点です：

| 次元 | 略称 | 説明 |
|------|------|------|
| 関連性 | REL | OpenClaw エコシステムとの関連度 |
| 互換性 | COM | 既存のワークフローとの統合のしやすさ |
| コミュニティ人気度 | TRC | ClawHub DL数、Reddit 言及数 |
| 価値提供 | VAL | 解決する問題の重要度と独自性 |
| メンテナンス | MNT | 更新頻度と開発者の活動状況 |
| 信頼性 | RLB | 安定性とエラー処理品質 |
| セキュリティ | SEC | 権限要件、コード品質、脆弱性履歴 |
| 学習価値 | LRN | 概念理解やスキル習得への貢献 |
