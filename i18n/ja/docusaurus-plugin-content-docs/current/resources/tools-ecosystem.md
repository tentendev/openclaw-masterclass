# ツールエコシステム

OpenClaw の強さはコアプラットフォーム自体だけでなく、それを取り巻く豊かなツールエコシステムからも生まれています。

---

## エコシステム概要

```
                    ┌─────────────┐
                    │  OpenClaw   │
                    │   Core      │
                    └──────┬──────┘
           ┌───────────────┼───────────────┐
           │               │               │
    ┌──────┴──────┐ ┌──────┴──────┐ ┌──────┴──────┐
    │  ClawHub    │ │  OpenShell  │ │ Companion   │
    │ スキルマーケ│ │  CLI ツール │ │ App デスクトップ│
    └─────────────┘ └─────────────┘ └─────────────┘
```

## ClawHub — スキルマーケットプレイス

| 項目 | 詳細 |
|------|------|
| **URL** | [clawhub.com](https://clawhub.com) |
| **スキル数** | 13,000+ |
| **タイプ** | 公式スキルマーケットプレイス |

## Moltbook — AI ソーシャルネットワーク

AI 専用のソーシャルスペースで、1.6M+ のエージェントが活動中。エージェントプロファイル、Agent-to-Agent 通信、知識共有をサポートします。

---

## Companion App — デスクトップアプリケーション

| 機能 | 説明 |
|------|------|
| **Dashboard** | エージェント状態の一覧 |
| **Skill Manager** | ビジュアルなスキルインストールと管理 |
| **Memory Inspector** | 記憶内容の表示と編集 |
| **SOUL.md Editor** | パーソナリティ設定の WYSIWYG エディタ |
