# 中国エコシステム

中国大陸は OpenClaw の最も急速に成長している市場の一つですが、同時に規制が最も複雑な市場でもあります。

:::danger 重要な法規注意
中国大陸では AI サービスの使用に厳格な法規要件があり、特に国有企業に対しては厳しいです。本ページの情報は参考のみで、法的助言を構成しません。
:::

---

## 中国市場概要

| 項目 | 状態 |
|------|------|
| **OpenClaw の中国での利用可能性** | 利用可能（オープンソース、自己デプロイ可能） |
| **ClawHub アクセス** | VPN が必要、または国内ミラーを使用 |
| **WeChat 公式統合** | 完了（Tencent 公式提携） |
| **推奨 LLM プロバイダー** | DeepSeek（VPN 不要） |
| **国有企業での使用** | 制限あり、コンプライアンス審査が必要 |

---

## DeepSeek — 中国大陸推奨 LLM

| メリット | 説明 |
|------|------|
| **VPN 不要** | API は中国大陸から直接アクセス可能 |
| **極めて低価格** | OpenAI の 1/10 ～ 1/30 |
| **中国語最適化** | 中国語シナリオに特化した最適化 |
| **オープンソースモデル** | ローカルデプロイ可能、データ完全制御 |

---

## コンプライアンス要件

| 法規 | 要件 |
|------|------|
| **生成型 AI サービス管理暫定弁法** | AI 生成コンテンツの審査メカニズムが必要、ログ6か月保持 |
| **個人情報保護法（PIPL）** | データ収集と使用方法のユーザーへの告知が必要 |
| **データセキュリティ法** | 重要データは中国国内に保存、データセキュリティ管理制度の確立 |
