# リサーチスキル (Research)

リサーチカテゴリのスキルにより、OpenClaw エージェントに「ウェブで調べる」能力が付与されます。基本的なウェブブラウジングから AI 駆動のインテリジェント検索まで、エージェントがリアルタイムの最新情報を取得できるようになります。

---

## #2 — Web Browsing

OpenClaw の最もコアなスキルの一つで、システムに直接組み込まれています。ウェブナビゲーション、インテリジェントなコンテンツ抽出、検索エンジン統合、JavaScript レンダリング、スクリーンショット機能、マルチタブ管理をサポートします。

## #4 — Tavily

AI エージェント向けに設計された検索エンジンで、セマンティック検索、ウェブクロール、ソース信頼度スコアリングを提供します。

## #15 — Felo Search

多言語最適化が特徴の AI 駆動検索ツールです。日本語・中国語の検索品質が Tavily より優れています。
