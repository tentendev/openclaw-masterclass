# メディアスキル (Media)

メディアカテゴリのスキルにより、OpenClaw エージェントにマルチメディア処理能力を付与します。

---

## #25 — Felo Slides

自然言語の説明からプロフェッショナルなプレゼンテーション（PPT/PDF）を生成します。テンプレートシステム、チャート生成、多言語サポート、PPTX/PDF/Google Slides へのエクスポートをサポートします。

## #27 — Spotify

Spotify の再生制御と音楽管理を行います。再生/一時停止/スキップ、楽曲検索、プレイリスト管理、シーンに応じた音楽推薦をサポートします。

## #32 — YouTube Digest

YouTube 動画から知識を抽出します。字幕取得、コンテンツサマリー、タイムスタンプノート、チャネル追跡をサポートします。

## #35 — Image Generation

複数の AI 画像生成サービスを通じて画像を作成します。DALL-E 3、Stable Diffusion、Midjourney API をサポートします。

## #39 — TweetClaw

X（旧 Twitter）のコンテンツ管理を行います。ツイートの閲覧、検索、下書き作成、トレンド追跡をサポートします。

## #40 — Voice / Vapi

OpenClaw エージェントに音声機能を追加します。音声入力（STT）、音声出力（TTS）、リアルタイム音声対話をサポートします。
