# スマートホームスキル (Smart Home)

スマートホームスキルにより、OpenClaw エージェントは画面の外に飛び出し、物理的なデバイスを制御します。

---

## #10 — Home Assistant

Home Assistant は最も人気のあるオープンソーススマートホームプラットフォームで、2000+ のデバイスブランドを統合しています。デバイス制御、シーン管理、自動化ルール、センサー読み取り、エネルギー監視をサポートします。

:::warning 物理的セキュリティ
Home Assistant はドアロック、監視カメラ、警報システムなどのセキュリティデバイスを制御できます。OpenClaw 用に**権限制限付き**の Home Assistant ユーザーを作成し、セキュリティに敏感なデバイスへのアクセスを除外してください。
:::

## #30 — Philips Hue

Hue Bridge API を通じた Philips Hue 照明システムの直接制御をサポートします。

## #36 — Elgato

Elgato 製品（Key Light、Light Strip）の制御をサポートします。ストリーマーやコンテンツクリエイター向けです。

## #42 — BambuLab 3D Printer

BambuLab 3D プリンターの監視と制御をサポートします。印刷進捗の確認、開始/一時停止/キャンセル、温度監視を行えます。
