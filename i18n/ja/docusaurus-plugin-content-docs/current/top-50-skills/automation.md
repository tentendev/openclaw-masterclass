# 自動化スキル (Automation)

自動化スキルにより、OpenClaw エージェントは「受動的な応答」から「能動的な実行」にレベルアップします。

---

## #8 — n8n

n8n はオープンソースのワークフロー自動化プラットフォームです。このスキルにより、エージェントが自然言語の説明からワークフローを作成し、500+ のサードパーティサービスと連携できます。

## #16 — Browser Automation

Web Browsing スキルよりも高度なブラウザ自動化ツールです。フォーム入力、クリック操作、ログイン自動化、フロー録画をサポートします。

## #37 — IFTTT

IFTTT Webhooks を通じた自動化をサポートします。700+ のサービスと連携可能な「ゼロインフラ」自動化方式です。

## #48 — Webhook Relay

OpenClaw エージェントが外部 Webhook 呼び出しを受信して反応できるようにします。「イベント駆動」エージェントモードを可能にします。
