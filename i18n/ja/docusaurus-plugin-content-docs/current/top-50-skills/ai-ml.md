# AI/ML スキル

AI/ML カテゴリのスキルにより、OpenClaw エージェントに「自己進化」と「知識構造化」の能力を付与します。

---

## #11 — Capability Evolver

エージェントが自動的に自身の能力を進化させることを可能にする、OpenClaw エコシステムで最も先進的なスキルの一つです。能力検出、自動学習、スキル提案、プロンプト最適化をサポートします。

## #22 — Ontology

エージェントがナレッジグラフを構築・照会できるようにします。エンティティ抽出、知識グラフ構築、推論クエリ、可視化をサポートします。

## #38 — RAG Pipeline

Retrieval-Augmented Generation (RAG) パイプラインを構築し、エージェントがカスタム知識ベースを照会できるようにします。

## #44 — Prompt Library

プロンプトテンプレートの管理と再利用ツールです。テンプレート保存、変数置換、バージョン管理、A/B テストをサポートします。

---

## AI/ML スキル学習パス

```
入門 → Prompt Library (#44)
       Prompt Engineering の基礎を理解
       ↓
中級 → RAG Pipeline (#38)
       Retrieval-Augmented Generation を学ぶ
       ↓
上級 → Ontology (#22)
       Knowledge Graph の概念を習得
       ↓
高度 → Capability Evolver (#11)
       Self-improving Agent を探求
```
