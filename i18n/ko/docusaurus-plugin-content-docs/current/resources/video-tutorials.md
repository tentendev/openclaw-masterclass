# 영상튜토리얼精選

比起閱讀문서，有些人更喜歡看영상學習。以下是社群精選的 OpenClaw 영상튜토리얼，按主題分類並附上평가。

:::tip 學習建議
建議先從「入門必看」區塊的영상開始，再根據你的興趣深入各主題。搭配 [學習路徑規劃](/docs/resources/learning-path) 一起使用效果最佳。
:::

## 입문 필수 시청

### VelvetShark — 28 分鐘完整 Walkthrough

| 項目 | 詳情 |
|------|------|
| **創作者** | VelvetShark |
| **長度** | 28 分鐘 |
| **語言** | 英文 |
| **適合程度** | 完全初學者 |
| **推薦指數** | ★★★★★ |

這是目前最受推薦的 OpenClaw 入門영상。VelvetShark 用 28 分鐘帶你從零開始完成以下步驟：

- OpenClaw 的핵심 개념介紹
- 다운로드與설치（包含常見問題排解）
- 首次설정與 API Key 配置
- 連接第一個커뮤니케이션平台
- SOUL.md 人格설정示範
- 설치第一個 ClawHub 스킬

:::info 추천하는 이유這部영상？
VelvetShark 的영상之所以被社群廣泛推薦，是因為他會展示所有真實的操作步驟，包含遇到錯誤時的排解過程。這比只展示「完美流程」的튜토리얼更有價值。
:::

## 영상 학습 팁

1. **調整播放速度**：熟悉的內容可以用 1.5x-2x 速度觀看
2. **邊看邊做**：準備好你的終端機，跟著영상一起操作
3. **做筆記**：記錄영상中提到的指令、설정值、注意事項
4. **重複觀看**：複雜的架構解說值得看兩遍以上
5. **加入社群**：在 [Discord](https://discord.gg/openclaw) 上討論영상中不懂的部分
