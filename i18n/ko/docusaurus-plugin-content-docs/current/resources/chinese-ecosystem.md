# 中國生態系統

중국是 OpenClaw 增長最快的市場之一，但同時也是法規最複雜的市場。從 Tencent WeChat 官方整合到國有엔터프라이즈使用限制，本頁涵蓋在중국使用 OpenClaw 所需了解的一切。

:::danger 중요 법규 주의
중국對 AI 服務的使用有嚴格的法規要求，特別是國有엔터프라이즈。本頁提供的資訊僅作為參考，不構成法律建議。在배포前，請務必諮詢專業法律顧問。
:::

---

## 중국 시장 개요

| 項目 | 狀態 |
|------|------|
| **OpenClaw 在中國的가용성** | 可用（開源軟體，可自行배포） |
| **ClawHub 存取** | 需要翻牆或使用國內映像 |
| **官方 WeChat 整合** | 已完成（Tencent 官方合作） |
| **推薦 LLM 提供者** | DeepSeek（無需翻牆） |
| **國有엔터프라이즈使用** | 有限制，需컴플라이언스審查 |
| **個人使用** | 無明確限制 |

## Tencent WeChat 整合

### 개요

Tencent 與 OpenClaw 的官方合作使得 WeChat（WeChat）成為 OpenClaw 支援的커뮤니케이션平台之一。這對중국사용자意義重大，因為 WeChat 是中國最普及的커뮤니케이션工具，擁有超過 13 億月活躍用戶。

### 整合功能

| 功能 | 支援狀態 | 說明 |
|------|---------|------|
| **個人聊天** | 完整支援 | Agent 可在一對一聊天中답변 |
| **群組聊天** | 完整支援 | Agent 可參與群組對話 |
| **공중 계정整合** | 部分支援 | 可作為공중 계정的後端智慧답변 |
| **미니 프로그램互動** | 計畫中 | 未來版本預計支援 |
| **WeChat Pay** | 不支援 | 涉及金融法規，不在計畫內 |
| **語音메시지** | 部分支援 | 需配合語音轉文字스킬 |

### 설정步驟

```yaml
channels:
  wechat:
    enabled: true
    app_id: "your_wechat_app_id"
    app_secret: "your_wechat_app_secret"
    token: "your_verification_token"
    encoding_aes_key: "your_aes_key"
    mode: "personal"  # 或 "official_account"
```

:::caution WeChat 설정 주의사항
WeChat 整合需要通過 Tencent 的개발자인증流程，個人계정與공중 계정的인증要求不同。공중 계정整合需要엔터프라이즈資質。
:::

### WeChat 整合的特殊限制

1. **메시지頻率限制**：WeChat 對메시지發送頻率有嚴格限制
2. **內容審查**：所有經由 WeChat 的메시지都會經過 Tencent 的內容審查系統
3. **데이터저장**：中國法規要求相關데이터必須저장在中國境內서버
4. **계정風險**：自動化過於激進可能導致 WeChat 계정被封禁

## DeepSeek — 推薦的중국 LLM

### 추천하는 이유 DeepSeek？

| 優勢 | 說明 |
|------|------|
| **無需翻牆** | API 在중국可直接存取 |
| **價格極低** | 僅為 OpenAI 的 1/10 到 1/30 |
| **中文優化** | 針對中文場景做過特別優化 |
| **本地컴플라이언스** | 符合중국데이터法規要求 |
| **開源模型** | 可在本地배포，데이터完全可控 |

### 與 OpenClaw 的整合

```yaml
llm:
  provider: deepseek
  api_key: sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
  model: deepseek-v3
  base_url: https://api.deepseek.com
```

### DeepSeek 模型比較

| 模型 | 適合場景 | 中文能力 | 價格等級 |
|------|---------|---------|---------|
| **DeepSeek-V3** | 通用對話、程式碼 | 優秀 | 極低 |
| **DeepSeek-R1** | 複雜推理 | 優秀 | 低 |
| **DeepSeek-Coder** | 程式碼專用 | 良好 | 極低 |

詳細的 API Key 申請流程請參考 [API Keys 取得가이드](/docs/resources/api-keys-guide)。

## 중국 사용 제한

### 國有엔터프라이즈限制

:::danger 국유기업 사용자 필독
중국國有엔터프라이즈（State-Owned Enterprises, SOEs）在使用 AI Agent 服務時有特殊限制。以下限制可能隨政策업데이트而變化，請定期確認最新法規。
:::

| 限制類別 | 說明 | 影響範圍 |
|---------|------|---------|
| **데이터存取限制** | Agent 不得存取涉及國家安全的內部데이터 | 所有國有엔터프라이즈 |
| **模型提供者限制** | 部分情況下僅允許使用國產 LLM | 敏感產業國企 |
| **배포位置限制** | 必須배포在中國境內서버 | 所有國有엔터프라이즈 |
| **審計要求** | 需定期제출 AI 使用報告 | 特定產業國企 |
| **스킬白名單** | 僅允許經심사的스킬 | 所有國有엔터프라이즈 |

### 一般엔터프라이즈컴플라이언스要求

即使非國有엔터프라이즈，在중국使用 OpenClaw 也需要注意：

1. **《生成式人工智能服務管理暫行辦法》**
   - AI 生成的內容必須符合社會主義核心價值觀
   - 需생성內容심사機制
   - 需保留使用로그至少六個月

2. **《個人信息保護法》（PIPL）**
   - 사용자的對話記錄屬於個人資訊
   - 需明確告知사용자데이터的收集與使用方式
   - 跨境傳輸데이터需額外評估

3. **《數據安全法》**
   - 需對處理的데이터進行安全等級分類
   - 重要데이터需在中國境內저장
   - 需생성데이터安全管理制度

### 컴플라이언스建議

| 措施 | 說明 | 優先級 |
|------|------|--------|
| 使用國產 LLM | 選擇 DeepSeek 等國產模型 | 高 |
| 境內배포 | 使用中國境內的雲端服務 | 高 |
| 로그保留 | 保留至少 6 個月的完整로그 | 高 |
| 內容過濾 | 생성內容심사機制 | 高 |
| 法律諮詢 | 聘請專業法律顧問 | 中 |
| 定期審計 | 每季度進行一次컴플라이언스審計 | 中 |

## 추천 중국 배포 아키텍처

```
┌─────────────────────────────────────────────┐
│  중국雲端서버（阿里雲 / 騰訊雲）      │
│                                             │
│  ┌──────────────┐   ┌──────────────┐       │
│  │  OpenClaw    │   │  DeepSeek    │       │
│  │  Core        │◄──│  API         │       │
│  └──────┬───────┘   └──────────────┘       │
│         │                                   │
│  ┌──────┴───────┐   ┌──────────────┐       │
│  │  Gateway     │◄──│  WeChat      │       │
│  │  (127.0.0.1) │   │  整合        │       │
│  └──────────────┘   └──────────────┘       │
│                                             │
│  ┌──────────────┐                           │
│  │  로그系統     │  ← 保留 6 個月以上       │
│  └──────────────┘                           │
└─────────────────────────────────────────────┘
```

:::tip 배포建議
使用阿里雲或騰訊雲的중국區域進行배포，確保滿足데이터在地化要求。Gateway 務必綁定到 `127.0.0.1`，透過雲端供應商的부하均衡器對外服務。
:::
