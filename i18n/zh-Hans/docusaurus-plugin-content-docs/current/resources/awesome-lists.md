# Awesome Lists 精选

「Awesome List」是开源社区中一种流行的资源整理方式，由社区成员共同维护、持续更新。以下是与 OpenClaw 相关的精选 Awesome Lists。

:::tip 什么是 Awesome List？
Awesome List 起源于 GitHub 上的 [awesome](https://github.com/sindresorhus/awesome) 项目，是一种以主题分类的高品质资源清单。星数越高，代表社区认可度越高。
:::

这是最完整的 OpenClaw 资源清单，涵盖以下分类：

- **Skills（技能）**：按类别分类的精选技能推荐

- **Tutorials（教程）**：文章、视频、交互教程
- **Tools（工具）**：第三方工具与集成
- **Configurations（配置）**：社区分享的最佳配置文件

## 如何贡献 Awesome List

想要将你发现的好资源加入 Awesome List？以下是一般的贡献流程：

1. **Fork** 该 Awesome List 的 GitHub repository
2. 在适当的分类下新增你推荐的资源
3. 确保格式一致（通常是 `- [名称](链接) - 简短说明`）
4. 提交 **Pull Request** 并说明推荐理由
5. 等待维护者审查与合并

### 提交品质标准

好的 Awesome List 项目通常具备以下特征：

- 资源是**免费**或**开源**的（付费资源需特别标注）
- 内容是**最新**的（过时的资源不适合）
- 有一定的**社区认可度**（Stars、下载量等）
- 附带清楚的**文件说明**

## 自行创建 Awesome List

如果你专精于某个 OpenClaw 相关领域，欢迎创建自己的 Awesome List：

```markdown
# Awesome OpenClaw [你的主题]

> 精选的 OpenClaw [主题] 资源清单

## 目录
- [工具](#工具)
- [教程](#教程)
- [示例](#示例)

## 工具
- [工具名称](链接) - 简短说明

## 教程
- [教程标题](链接) - 简短说明
```
