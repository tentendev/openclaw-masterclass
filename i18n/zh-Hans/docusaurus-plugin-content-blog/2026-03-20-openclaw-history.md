# OpenClaw 的故事：从 Clawdbot 到 250K Stars

每一个伟大的开源项目背后都有一段精彩的故事。OpenClaw 的旅程——从一个个人 Side Project 到拥有 250,000+ GitHub Stars 的现象级项目——是开源精神最好的诠释。

<!-- truncate -->

## 起源：Clawdbot（2024 年中）

故事要从 **Peter Steinberger** 说起。Steinberger 是 iOS 开发社区中的知名人物，曾创办 PSPDFKit（后来的 Nutrient），是一位连续创业者和开源倡导者。

2024 年中，Steinberger 开始了一个个人实验项目——**Clawdbot**。最初的构想很简单：构建一个能够连接多种通讯平台的 AI 聊天机器人框架。他希望这个框架能够：

- 在一个统一的界面下管理多个通讯平台
- 支持插件式的功能扩展
- 让非技术人员也能轻松部署 AI 助手

Clawdbot 这个名字来自于"Claw"（爪子）和"Bot"的结合——后来龙虾的形象便由此而来。

## 演化：Moltbot（2024 年底）

随着项目的发展，Steinberger 意识到原始设计需要大幅重构。就像龙虾需要蜕壳才能成长一样，项目也经历了一次"蜕变"（Molt）。

2024 年底，Clawdbot 重构为 **Moltbot**，带来了全新的架构：

- **三层架构设计**：Gateway（连接层）→ Agent（代理层）→ Skill（功能层）
- **WebSocket 实时通讯**：取代了原本的 REST polling 机制
- **Skill Marketplace 雏形**：开发者可以分享自己的 Skills

Moltbot 在小范围的开发者社区中引起了关注，GitHub Stars 在短短几周内突破了 10,000。

## 转折点：重新命名为 OpenClaw（2026 年 1 月）

2026 年 1 月，项目迎来了最重要的里程碑之一——正式更名为 **OpenClaw**。

这次更名不仅仅是换一个名字，更代表了项目方向的重大转变：

1. **全面开源治理**：成立了 OpenClaw Foundation，引入社区治理模式
2. **企业级功能**：新增了 RBAC（角色权限控制）、审计日志、SSO 集成
3. **多语言支持**：除了原本的 TypeScript/JavaScript，新增了 Python、Go、Rust、Java SDK
4. **官方 Skill Marketplace**：超过 13,000 个社区贡献的 Skills

更名为 OpenClaw 后，项目的增长速度令人难以置信。在短短两个月内，GitHub Stars 从 50,000 飙升到 150,000。

## Peter Steinberger 加入 OpenAI（2026 年 2 月）

2026 年 2 月，一则震动开源社区的消息传出——**Peter Steinberger 加入了 OpenAI**。

Steinberger 将在 OpenAI 领导一个新的团队，专注于"AI Agent 基础设施"。这个决定引起了社区的广泛讨论：

> "这对 OpenClaw 意味着什么？"

Steinberger 在公告中明确表示：

- OpenClaw 将继续作为独立的开源项目运作
- OpenClaw Foundation 已经建立了完善的社区治理机制
- 他将以顾问的身份继续参与 OpenClaw 的发展方向

事实证明，这个决定反而为 OpenClaw 带来了更多的关注和资源。OpenAI 也成为了 OpenClaw Foundation 的企业赞助商之一。

## 250K Stars 里程碑（2026 年 3 月）

2026 年 3 月，OpenClaw 正式突破了 **250,000 GitHub Stars**，成为 GitHub 上增长最快的开源项目之一。

这个数字背后代表的是：

- **活跃贡献者**：超过 2,000 位来自全球的贡献者
- **Skill 生态系统**：13,000+ 社区 Skills，涵盖生产力、开发工具、社区管理等类别
- **平台集成**：支持 20+ 通讯平台（Slack、Discord、Teams、LINE、Telegram、WhatsApp 等）
- **企业采用**：数百家企业在生产环境中使用 OpenClaw
- **语言支持**：5 种编程语言的官方 SDK

## 时间线总览

| 2024 年底 | 重构为 Moltbot，引入三层架构 |
| 2025 年 | Moltbot 持续成长，社区逐渐壮大 |
| 2026 年 1 月 | 正式更名为 OpenClaw，成立 Foundation |
| 2026 年 2 月 | Steinberger 加入 OpenAI |
| 2026 年 3 月 | 突破 250K GitHub Stars |

## 展望未来

OpenClaw 的故事还远没有结束。社区正在积极开发的功能包括：

- **OpenClaw Cloud**：官方托管服务，一键部署
- **Skill Studio**：可视化的 Skill 开发工具
- **Agent Orchestration**：多 Agent 协作框架
- **Edge Deployment**：支持在边缘设备上运行 OpenClaw

从一个人的 Side Project 到全球现象级开源项目，OpenClaw 的旅程证明了一件事：**一个好的想法加上开放的社区，可以改变世界。**
