# OpenClaw 的故事：從 Clawdbot 到 250K Stars

每一個偉大的開源專案背後都有一段精彩的故事。OpenClaw 的旅程——從一個個人 Side Project 到擁有 250,000+ GitHub Stars 的現象級專案——是開源精神最好的詮釋。

<!-- truncate -->

## 起源：Clawdbot（2024 年中）

故事要從 **Peter Steinberger** 說起。Steinberger 是 iOS 開發社群中的知名人物，曾創辦 PSPDFKit（後來的 Nutrient），是一位連續創業者和開源倡導者。

2024 年中，Steinberger 開始了一個個人實驗專案——**Clawdbot**。最初的構想很簡單：建立一個能夠連接多種通訊平台的 AI 聊天機器人框架。他希望這個框架能夠：

- 在一個統一的介面下管理多個通訊平台
- 支援插件式的功能擴展
- 讓非技術人員也能輕鬆部署 AI 助手

Clawdbot 這個名字來自於「Claw」（爪子）和「Bot」的結合——後來龍蝦的意象便由此而來。

## 演化：Moltbot（2024 年底）

隨著專案的發展，Steinberger 意識到原始設計需要大幅重構。就像龍蝦需要脫殼才能成長一樣，專案也經歷了一次「蛻變」（Molt）。

2024 年底，Clawdbot 重構為 **Moltbot**，帶來了全新的架構：

- **三層架構設計**：Gateway（連接層）→ Agent（代理層）→ Skill（功能層）
- **WebSocket 即時通訊**：取代了原本的 REST polling 機制
- **Skill Marketplace 雛形**：開發者可以分享自己的 Skills

Moltbot 在小範圍的開發者社群中引起了關注，GitHub Stars 在短短幾週內突破了 10,000。

## 轉捩點：重新命名為 OpenClaw（2026 年 1 月）

2026 年 1 月，專案迎來了最重要的里程碑之一——正式更名為 **OpenClaw**。

這次更名不僅僅是換一個名字，更代表了專案方向的重大轉變：

1. **全面開源治理**：成立了 OpenClaw Foundation，引入社群治理模式
2. **企業級功能**：新增了 RBAC（角色權限控制）、審計日誌、SSO 整合
3. **多語言支援**：除了原本的 TypeScript/JavaScript，新增了 Python、Go、Rust、Java SDK
4. **官方 Skill Marketplace**：超過 13,000 個社群貢獻的 Skills

更名為 OpenClaw 後，專案的成長速度令人難以置信。在短短兩個月內，GitHub Stars 從 50,000 飆升到 150,000。

## Peter Steinberger 加入 OpenAI（2026 年 2 月）

2026 年 2 月，一則震撼開源社群的消息傳出——**Peter Steinberger 加入了 OpenAI**。

Steinberger 將在 OpenAI 領導一個新的團隊，專注於「AI Agent 基礎設施」。這個決定引起了社群的廣泛討論：

> 「這對 OpenClaw 意味著什麼？」

Steinberger 在公告中明確表示：

- OpenClaw 將繼續作為獨立的開源專案運作
- OpenClaw Foundation 已經建立了完善的社群治理機制
- 他將以顧問的身份繼續參與 OpenClaw 的發展方向

事實證明，這個決定反而為 OpenClaw 帶來了更多的關注和資源。OpenAI 也成為了 OpenClaw Foundation 的企業贊助商之一。

## 250K Stars 里程碑（2026 年 3 月）

2026 年 3 月，OpenClaw 正式突破了 **250,000 GitHub Stars**，成為 GitHub 上成長最快的開源專案之一。

這個數字背後代表的是：

- **活躍貢獻者**：超過 2,000 位來自全球的貢獻者
- **Skill 生態系統**：13,000+ 社群 Skills，涵蓋生產力、開發工具、社群管理等類別
- **平台整合**：支援 20+ 通訊平台（Slack、Discord、Teams、LINE、Telegram、WhatsApp 等）
- **企業採用**：數百家企業在生產環境中使用 OpenClaw
- **語言支援**：5 種程式語言的官方 SDK

## 時間線總覽

| 2025 年 | Moltbot 持續成長，社群逐漸壯大 |
| 2026 年 1 月 | 正式更名為 OpenClaw，成立 Foundation |
| 2026 年 2 月 | Steinberger 加入 OpenAI |
| 2026 年 3 月 | 突破 250K GitHub Stars |

## 展望未來

OpenClaw 的故事還遠沒有結束。社群正在積極開發的功能包括：

- **OpenClaw Cloud**：官方託管服務，一鍵部署
- **Skill Studio**：視覺化的 Skill 開發工具
- **Agent Orchestration**：多 Agent 協作框架
- **Edge Deployment**：支援在邊緣設備上運行 OpenClaw

從一個人的 Side Project 到全球現象級開源專案，OpenClaw 的旅程證明了一件事：**一個好的想法加上開放的社群，可以改變世界。**
