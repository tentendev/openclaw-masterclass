# 影片教學精選

比起閱讀文件，有些人更喜歡看影片學習。以下是社群精選的 OpenClaw 影片教學，按主題分類並附上評價。

:::tip 學習建議
建議先從「入門必看」區塊的影片開始，再根據你的興趣深入各主題。搭配 [學習路徑規劃](/docs/resources/learning-path) 一起使用效果最佳。
:::

| **長度** | 28 分鐘 |
| **語言** | 英文 |
| **適合程度** | 完全初學者 |
| **推薦指數** | ★★★★★ |

這是目前最受推薦的 OpenClaw 入門影片。VelvetShark 用 28 分鐘帶你從零開始完成以下步驟：

- OpenClaw 的核心概念介紹
- 下載與安裝（包含常見問題排解）
- 首次設定與 API Key 配置
- 連接第一個通訊平台

:::info 為什麼推薦這部影片？
VelvetShark 的影片之所以被社群廣泛推薦，是因為他會展示所有真實的操作步驟，包含遇到錯誤時的排解過程。這比只展示「完美流程」的教學更有價值。
:::

## 影片學習技巧

1. **調整播放速度**：熟悉的內容可以用 1.5x-2x 速度觀看
2. **邊看邊做**：準備好你的終端機，跟著影片一起操作
3. **做筆記**：記錄影片中提到的指令、設定值、注意事項
4. **重複觀看**：複雜的架構解說值得看兩遍以上
5. **加入社群**：在 [Discord](https://discord.gg/openclaw) 上討論影片中不懂的部分
